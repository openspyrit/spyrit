class spyrit.restructured.Updated_Had_Dcan.Denoise_layer(*args: Any, **kwargs: Any)[source]

Bases: Module

Applies a transformation to the incoming data: :math:`y = A^2/(A^2+x) `

Args:

in_features: size of each input sample

Shape:

  • Input: \((N, *, H_{in})\) where \(*\) means any number of additional dimensions and \(H_{in} = \text{in\_features}\)

  • Output: \((N, *, H_{in})\).

Attributes:
weight: the learnable weights of the module of shape

\((\text{out\_features}, 1)\). The values are initialized from \(\mathcal{U}(-\sqrt{k}, \sqrt{k})\), where \(k = \frac{1}{\text{in\_features}}\)

Examples::
>>> m = Denoise_layer(30)
>>> input = torch.randn(128, 30)
>>> output = m(input)
>>> print(output.size())
torch.Size([128, 30])

Preprocess the measurements acquired using positive (shifted) patterns corrupted by Poisson noise

The output value of the layer with input size \((B*C, M)\) can be described as:

\[\text{out}((B*C)_i, M_j}) = 2*\text{input}((B*C)_i, M_j}) - \sum_{k = 1}^{M-1} \text{input}((B*C)_i, M_k})\]

The output size of the layer is \((B*C, M)\), which is the imput size

The output size of the layer is \((B*C, M)\)
