spyrit.core.prep.SplitPoisson

class spyrit.core.prep.SplitPoisson(alpha: float, meas_op)[source]

Bases: Module

Preprocess the raw data acquired with a split measurement operator assuming Poisson noise. It also compensates for the affine transformation applied to the images to get positive intensities.

It computes \(m = \frac{y_{+}-y_{-}}{\alpha} - H1\) and the variance \(var = \frac{2(y_{+} + y_{-})}{\alpha^{2}}\), where \(y_{+} = H_{+}x\) and \(y_{-} = H_{-}x\) are obtained using a split measurement operator (see spyrit.core.LinearSplit), \(\alpha\) is the image intensity, and 1 is the all-ones vector.

Args:

alpha (float): maximun image intensity \(\alpha\) (in counts)

meas_op: measurement operator (see meas)

Example:
>>> H = np.random.random([400,32*32])
>>> meas_op =  LinearSplit(H)
>>> split_op = SplitPoisson(10, meas_op)
Example 2:
>>> Perm = np.random.random([32,32])
>>> meas_op = HadamSplit(400, 32,  Perm)
>>> split_op = SplitPoisson(10, meas_op)
