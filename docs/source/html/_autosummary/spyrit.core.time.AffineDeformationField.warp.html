spyrit.core.time.AffineDeformationField.warp

AffineDeformationField.warp(img: tensor, t0: float, t1: float = None, n_frames: int = None, fps: float = None, mode: str = 'bilinear') → tensor[source]

Warps an image according to the time interpolation parameters.

Similarly to the method DeformationField.warp() from the parent class, it warps a single image according to the inverse deformation field \(v\) contained in the attribute inverse_grid_frames, between the times \(t0\) and \(t1\). The number of frames in the animation is given by either \(n\_frames\), or \(fps\) if \(n\_frames\) is None.

Args:

img (torch.tensor): Image to deform of shape \((c,Nx,Ny)\), where \(c\) is the number of channels, and \(Nx\) and \(Ny\) are the number of pixels along the x-axis and y-axis respectively. The number of channels is usually 1 (grayscale) or 3 (color), if not a warning is raised.

t0 (float): Start time of the animation.

t1 (float): End time of the animation. If None, a single frame is warped.

n_frames (int): Number of frames in the animation. If None, fps is used to compute the number of frames.

fps (float): Number of frames per second. If None, n_frames is used, or a single frame is warped.

mode (str, optional): The interpolation mode to use. It is directly passed to the function torch.nn.functional.grid_sample. It must be one of the following: ‘nearest’, ‘bilinear’, ‘bicubic’. Defaults to ‘bilinear’.

Returns:

output (torch.tensor): The deformed batch of images of shape \((n\_frames,c,Nx,Ny)\), where each image is deformed according to the inverse deformation field \(v\) contained in the attribute inverse_grid_frames.

Example 1: Rotate a single image by 90° counter-clockwise
>>> def v(t):
...     return torch.tensor([[0, -1, 0], [1, 0, 0], [0, 0, 1]])
>>> field = AffineDeformationField(v, align_corners=False)
>>> image = torch.tensor([[[0. , 0.3],
                           [0.7, 1. ]]])
>>> deformed_image = field.warp(image, 0)
>>> print(deformed_image)
tensor([[[[0.3000, 1.0000],
          [0.0000, 0.7000]]]])
Example 2: Animate an image between t0=0 and t1=1, with 5 frames, with counter-clockwise rotation
>>> def s(t):
...     return np.sin(2*np.pi*t)
>>> def c(t):
...     return np.cos(2*np.pi*t)
>>> def v(t):
...     return torch.tensor([[c(t), -s(t), 0], [s(t), c(t), 0], [0, 0, 1]])
>>> field = AffineDeformationField(v, align_corners=False)
>>> image = torch.tensor([[[0. , 0.3],
                           [0.7, 1. ]]])
>>> deformed_image = field.warp(image, 0, 1, 5)
>>> print(deformed_image)
tensor([[[[0.0000, 0.3000],
          [0.7000, 1.0000]]],
[[[0.3000, 1.0000],

[0.0000, 0.7000]]],

[[[1.0000, 0.7000],

[0.3000, 0.0000]]],

[[[0.7000, 0.0000],

[1.0000, 0.3000]]],

[[[0.0000, 0.3000],

[0.7000, 1.0000]]]])
