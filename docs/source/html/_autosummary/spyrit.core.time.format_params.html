spyrit.core.time.format_params

spyrit.core.time.format_params(self, t0: float, t1: float, n_frames: int, fps: float) → tuple[source]

Returns the corrected parameters t0, t1 and n_frames

Returns the parameters t0, t1 and n_frames in a format that can be used to animate an image in multiple frames or warp an image in a single frame.

Note

The parameters n_frames and fps are mutually exclusive. If both are given, n_frames is used. If both are None, a single frame is warped.

Note

If fps is given and n_frames is None, the end time t1 is truncated to the closest lowest multiple of \(1/\text{fps}\). The number of frames is then computed as usual between t0 and the truncated t1.

Args:

t0 (float): Start time of the animation.

t1 (float): End time of the animation. If None, a single frame is warped.

n_frames (int): Number of frames in the animation. If None, a single frame is warped.

fps (float): Number of frames per second. If None, a single frame is warped.

Returns:

(t0, t1, n_frames) (tuple): Where t0 (float) and t1 (float) are the start and end time of the animation respectively, and n_frames (int) is the number of frames in the animation.

The output follows this pattern:

Is :attr:`t1` :attr:`None`?
├── Yes
│    └── Are :attr:`n_frames` and :attr:`fps` :attr:`None`?
│         ├── Yes ──> :attr:`(t0, t0, 1)`: a single frame is warped.
│         └── No ──> :attr:`ValueError` (conflicting parameters).
└── No
    └── Is :attr:`n_frames` :attr:`None`?
        ├── Yes
        │    └── Is :attr:`fps` :attr:`None`?
        │         ├── Yes ──> :attr:`ValueError` (conflicting parameters).
        │         └── No ─┬─ :attr:`n_frames = int(np.floor(1 + (t1-t0) * fps))`
        │                 └──> :attr:`(t0, t0 + (n_frames-1) / fps, n_frames)`
        └── No ──> :attr:`(t0, t1, n_frames)`
Example 1: Warp a single image at t0=0, with 1 frame
>>> field = AffineDeformationField(None, align_corners=False)
>>> field.format_params(0)
(0, 0, 1)
Example 2: Warp a single image between t0=0 and t1=1, with 11 frames
>>> field = AffineDeformationField(None, align_corners=False)
>>> field.format_params(0, 1, 11)
(0, 1, 11)
Example 3: Warp a single image between t0=0 and t1=1 at 24 fps
>>> field = AffineDeformationField(None, align_corners=False)
>>> field.format_params(0, 1, None, 24)
(0, 1, 25)
Example 4: Provide n_frames and fps
>>> field = AffineDeformationField(None, align_corners=False)
>>> field.format_params(0, 1, 11, 24)
(0, 1, 11)
Example 5: Provide conflicting parameters
>>> field = AffineDeformationField(None, align_corners=False)
>>> field.format_params(0, 1, None, None)
ValueError: Unable to animate multiple frames: t1 (1) was given, but fps and n_frames were not given.
Example 6: Provide conflicting parameters
>>> field = AffineDeformationField(None, align_corners=False)
>>> field.format_params(0, None, 11, 24)
ValueError: Unable to animate multiple frames: t1 was not given, but n_frames (11) or fps (24) or were given.
